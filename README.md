# Oasis Infobyte Web Development Internship (OIBSIP)

This repository contains the projects I developed during my **Web Development Internship at Oasis Infobyte**.  
The internship focuses on building practical frontend development skills by creating real-world projects using modern web technologies.

Each task demonstrates different aspects of **HTML, CSS, JavaScript, and responsive UI design**.

---

## 🚀 Internship Tasks

### 🔹 Task 1 – Landing Page -Completed
A modern coffee-themed landing page inspired by Starbucks.

**Features**
- Responsive navigation bar using **Flexbox**
- Smooth animations using **AOS (Animate On Scroll) library**
- Interactive hover effects on links and buttons
- Clickable coffee cup thumbnails with **rotation and opacity animations**
- Alternating image and content sections using **Flexbox**
- Structured footer built using **CSS Grid**

**Technologies Used**
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- AOS Animation Library

### 🔹 Task 2 – Personal Portfolio Website

This project is a **Personal Portfolio Website** developed as **Task 2 of the Oasis Infobyte Web Development Internship**.

The portfolio showcases my **skills, projects, and contact information** in a modern and interactive layout.

## 🚀 Features

* Responsive **Navbar with navigation links**
* **Hero section with live space animation** using tsParticles
* **Typing text animation** for dynamic introduction
* Profile image with **floating animation and glowing border**
* **Skills section with interactive tilt effect cards**
* **Projects section** displaying recent projects with hover effects
* **Contact form** for easy communication

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* tsParticles (for background animation)
* jQuery Tilt (for card tilt effect)

This project helped me practice **frontend development, UI design, and interactive web animations**.
