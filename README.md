# Oasis Infobyte Web Development Internship (OIBSIP)

This repository contains the projects I developed during my **Web Development Internship at Oasis Infobyte**.  
The internship focuses on building practical frontend development skills by creating real-world projects using modern web technologies.

Each task demonstrates different aspects of **HTML, CSS, JavaScript, and responsive UI design**.

---

## 🚀 Internship Tasks

### 🔹 Task 1 – Landing Page -Completed
A modern coffee-themed landing page inspired by Starbucks.

**Features**
- Responsive navigation bar using **Flexbox**
- Smooth animations using **AOS (Animate On Scroll) library**
- Interactive hover effects on links and buttons
- Clickable coffee cup thumbnails with **rotation and opacity animations**
- Alternating image and content sections using **Flexbox**
- Structured footer built using **CSS Grid**

**Technologies Used**
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- AOS Animation Library
